스마트폰 홈 화면에 앱으로 설치하기

별도 앱스토어 다운로드 없이 브라우저에서 바로 앱으로 등록할 수 있습니다.

아이폰 (iOS / Safari)
  1. Safari 브라우저에서 현재 페이지 접속
  2. 하단 중앙의 공유 버튼터치
  3. 메뉴에서 '홈 화면에 추가' 선택
  4. 우측 상단 [추가] 선택 시 설치 완료
안드로이드 (Android / Chrome)
  1. Chrome 브라우저에서 현재 페이지 접속
  2. 우측 상단 점 3개 메뉴(⋮) 터치
  3. '앱 설치' 또는 '홈 화면에 추가' 선택
  4. 팝업 창에서 [설치] 선택 시 완료
